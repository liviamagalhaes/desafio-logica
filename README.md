## Sobre o Repositório

Este repositório é o meu espaço pessoal de estudos e prática para aprimorar o meu **Pensamento Analítico** e a **Lógica de Programação**. O objetivo é consolidar os fundamentos que regem qualquer linguagem de desenvolvimento, transformando desafios em algoritmos eficientes.

Acredito que entender a lógica por trás do código é o primeiro passo para me tornar uma desenvolvedora completa.

## Conceitos Explorados

Durante os meus estudos, estou aplicando os seguintes conceitos:

* [x] Variáveis e Constantes
* [x] Operadores Aritméticos e Lógicos
* [x] Estruturas de Decisão (`if`, `else`)
* [x] Laços de Repetição (`for`, `while`)
* [x] Funções
* [x] Arrays e Matrizes (Estruturas de Dados Básicas)
* [x] Classes e Objetos (Introdução à Orientação a Objetos)
